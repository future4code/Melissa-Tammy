Projeto Future4 - Seleção de alunos para o curso Full Stack


Objetivo

O projeto consiste na criação de um site de apenas uma página utilizando HTML, CSS e Javascript. O candidato deveria seguir um layout disponibilizado pela F4, e seguir alguns requisitos.

    Requisitos

  • seguir, da melhor forma que conseguir, o layout que está no arquivo wireframe.pdf
  • campo de input de texto no final da página e um botão com o texto "Enviar"
  • Ao clicar nesse botão, um alerta deve ser apresentado na página com o texto "O email usuario@future4.com.br foi enviado", sendo que o email deve conter o texto digitado pelo usuário da página.

    • Desafios extras

  • Preparar o site para ser apresentado em telas mobile
  • Realizar a verificação de email válido no texto digitado pelo usuário no campo "email"
  • Quando o usuário clicar no botão enviar, a página deve ser atualizada para incluir um texto no formato "Bem vindo usuario@future4.com.br!"
