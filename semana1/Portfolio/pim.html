Projeto Integrado Multidisciplinar


Objetivo

Os conhecimentos adquiridos ao longo do primeiro semestre do Curso Superior Tecnológico em Análise e Desenvolvimento de Sistemas da Universidade Paulista foram utilizados para o desenvolvimento de um software, em linguagem C e modo console, para o controle de uma pizzaria fictícia; apresentando o contexto de uso, regras de negócio, fluxogramas, elaboração de um diagrama de redes de computadores e Cálculos matemáticos. Sendo utilizado a Engenharia de Software em todos os processos para o desenvolvimento adequado e eficaz.

Disciplinas contempladas:

  • Linguagem e Técnicas de Programação
  • Engenharia de Software I
  • Fundamentos Redes Dados e Comunicação
  • Matemática para Computação

Comandos utilizados

  • Condicionais e retição
    validação de login
  • Switch Case
    seleção de funcionalidades
  • Registro
    junção de dados de pedidos
  • alocação de memória dinâmica
    armazenar os registros gerados pelos cadastros de clientes, funcionários, pedidos e produtos
  • Malha de repetição
    Ordenar dados
