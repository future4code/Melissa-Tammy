Projeto Site Administrativo


Objetivo

O projeto consiste na criação de um site administrativo para uma loja, utilizando HTML e CSS.

    Contendo as seguintes páginas:

  • Página inicial
    Essa página terá o nome da empresa, uma mensagem de boas vindas, e um menu com links para outras páginas.
  • Controle de estoque
    Essa página deve ter uma tabela com os produtos da loja, mostrando uma foto, uma descrição, o preço e a quantidade disponível.
  • Adicionar produto
    Essa página deve conter um formulário para adicionar produtos. Ele deve ter os mesmos campos que aparecem no controle de estoque.
  • Funcionários
    Essa página deve mostrar uma lista dos funcionários, com o nome em negrito e a data de nascimento de cada um.
  • Página de Venda
    Deve conter um formulário no qual o vendedor coloque cada produto que o cliente está comprando. Crie uma lista com esses produtos. Coloque, também, as opções de pagamento: dinheiro ou cartão.
